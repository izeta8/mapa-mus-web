# Mapa Mus Color Palette - Basque Inspired (Green Primary)

This color palette is inspired by the Basque flag (Ikurriña) and refined for a modern, clean, and user-friendly mobile application UI. It provides a comprehensive set of colors for a **light theme only**, ensuring consistency and a strong visual identity rooted in the Basque aesthetic.

**This version uses Green as the primary color for a positive and encouraging user experience.**

> Note: Take into account that the UI must have a modern look like shadcn.
